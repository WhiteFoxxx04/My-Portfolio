Overview: Bug Tracker

Bug Tracker is an application to track errors in software. It was created by Asgharali Khan. Bug Tracker uses a ticketing system to record bugs or other issues in the software development process on a per-project basis. It implements user and role-level security to ensure only authorized users can access tickets and projects.

Technologies Used:
